# Интеграция VotingGuide с Your Estimate

Этот план описывает шаги по связке интерактивной таблицы подсказок с основной логикой голосования.

## Шаги реализации

### 1. Подготовка интерфейса VotingGuide
- [ ] Обновить `VotingGuideProps` в `components/voting-guide.tsx`, чтобы принимать callback `onVote: (vote: string) => void`.
- [ ] Добавить `useEffect`, который следит за изменением `suggestedSpIndex` или `isAiMode`.
- [ ] Когда `suggestedSpIndex` меняется и не равен -1, вызывать `onVote` с текущим значением SP.

### 2. Связка в Room Page
- [ ] В `app/room/[id]/page.tsx` передать функцию `handleVote` в компонент `VotingGuide`.
- [ ] Убедиться, что передаваемое значение SP из гайда соответствует типу `VoteValue` (валидация).

### 3. Проверка и тесты
- [ ] Проверить, что при кликах в таблицу голос улетает на сервер.
- [ ] Проверить, что в режиме AI Mode улетают "дисконтированные" Story Points.
- [ ] Убедиться, что ручной клик по карточкам все еще работает и имеет приоритет (последний клик побеждает).
